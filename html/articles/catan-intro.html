A Scala Project

I was recently gifted a Catan expansion pack (the Cities & Knights expansion), and I've thought to myself "wouldn't it be nice to learn the rules interactively, before playing with guests who would also have to learn the rules at the same time?". Well, instead of just using the official app Catan Universe, I've decided to use this as motivation to build my own online version of Catan.

My other motivation in starting on this project is to learn how Scala.js works. Scala is a language that I've writing in more than any other language, so it's only natural that I also try out Scala.js as well. Scala.js does seem strange to start a project with in 2023 when considering the rich ecosystem that JavaScript and TypeScript provide, but my motivation here is to try out programming paradigms that I'm unfamiliar with.

This project will encompass many posts, which I will link here once completed. To see the current state of the project, try out the live version here or check out the code at github.com/david-wiles/catan
