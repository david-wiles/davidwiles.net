One of first hits of the arcade era was Atari’s Asteroids, and it is still one of the most well known games of all time. For this reason, I’ve decided to remake it in a web-based version.

You can play the game here. The arrow keys control the spaceship, f to shoot and s for shield.

You can find the source code on github here: https://github.com/david-wiles/asteroids-ts.
